# Tuning

Shape and experiment with your instrument's tuning.

## Tuning toolbar
   - Delete/Save
   - Presets / Saved Tunings
   - Cents / Ratio toggle
   - Add element
## Tuning dashboard
   ### Default: tuning overview
   Displays a table of the elements. Each column contains Frequency, Cents, and Ratio.
   ### Selected: element overview
       - Properties
       - Style
